    Action / Tasklist manager

    Inspired by the great book Getting Things Done by David Allan. This is a program I wrote because I badly needed it in order to organize my own time. I failed to find anything, on any platform, that satisfied my requirements:

    • Free
    • Preferably Open Source
    • Designed after the principles in "Getting Things Done"
    • Safe; no unnecessary app privileges, no enforced integration with Google (or any other) services in order to work.
    • Fast and efficient to work with
    • Safeguards against losing data by accidents in stressful environments (for example by canceling a half edited task by hitting the Android back button).
    • Professionally designed and implemented (I'm just too tired with amateurish implemented "apps").

    These requirements became my blueprint when I built VikingGTD. And unlike the majority of today’s "app" writers, I am a professional software developer with more than 25 years of experience in software design and implementation. VikingGTD is free of eye candy and animations – but in stead very fast and responsive, also when the number of tasks grows during real use by busy people.

    The current version is optimized for large handsets like the Samsung Galaxy III and Note II. A tablet enabled version is in the works and will be released soon. Synchronization between devices will also be released soon. Since I'm using the app myself to organize most aspects of my work and my life, I'm rapidly implementing new features.

    Features

    1. Any number of actions in one of any number of lists.
    2. Each action can be configured with when, where, how and at what priority. These parameters can then be used to easily sort trough and re-organize the actions.
    3. Search in lists and actions.
    4. Fast access to main functions and frequently used features
    5. Easy to schedule an action to any time, date, week, month, or year.
    6. Easy and fast to reschedule one or more activities
    7. No built in limitations, no nagging about money, no ads.
    8. No phone home, sneaking in private data or other misbehavior.
    9. Import data from the TaskList app if you already use that to organize your life.
    10. Export and import the data to/from xml for backup and transfer to other devices.
